# Контрольная работа (2 семестр, заочная форма) — вариант 7

Зачётная книжка 257466 → последние три цифры 466 → 4+6+6 = 16 → 1+6 = **7**.

## Задача
На ввод поступает строка символов. Преобразовать всю строку к верхнему регистру,
если заглавных символов больше, чем строчных, и наоборот. В случае равного
количества — преобразовать к строчному виду.

Пример: `ABccAAr → ABCCAAR`, `abCCaaR → abccaar`.

## Решение
Файл [`CaseConverter.java`](CaseConverter.java). Программа считает количество
заглавных и строчных букв; если заглавных строго больше — приводит строку к
верхнему регистру, иначе (в том числе при равенстве) — к нижнему.

## Запуск
```
javac -encoding UTF-8 CaseConverter.java
java CaseConverter
```

## Примеры
| Ввод      | Вывод   |
|-----------|---------|
| ABccAAr   | ABCCAAR |
| abCCaaR   | abccaar |
| aBcD      | abcd    |
| HELLO     | HELLO   |
