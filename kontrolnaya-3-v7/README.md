# Контрольная работа (2 семестр, заочная форма) — вариант 7

Зачётная книжка 257466 → последние три цифры 466 → 4+6+6 = 16 → 1+6 = **7**.

## Задача
Требуется написать программу по расшифровке алгоритма Цезаря. Для формирования
частотного словаря используется текст с https://fish-text.ru.

## Решение
Файл [`CaesarDecrypt.java`](CaesarDecrypt.java). Алгоритм:
1. загружает русский текст с fish-text.ru и строит эталонные частоты букв
   (ответ приходит в формате JSON, экранированные символы декодируются);
2. перебирает все возможные сдвиги шифра Цезаря (0–31);
3. для каждого сдвига сравнивает распределение букв расшифрованного текста
   с эталонным (сумма квадратов отклонений частот);
4. выбирает сдвиг с наименьшим отклонением и выводит его и расшифрованный текст.

При отсутствии сети используется встроенный запасной текст.

## Запуск
```
javac -encoding UTF-8 CaesarDecrypt.java
java CaesarDecrypt
```

## Пример работы
```
Зашифрованный текст: бкыьшьчеу кчкхтс щшсмшхйпь ... лэфм ъэыыфшнш йсефк
Найденный сдвиг: 22
Расшифровка: частотный анализ позволяет восстановить исходный текст ... букв русского языка
```
