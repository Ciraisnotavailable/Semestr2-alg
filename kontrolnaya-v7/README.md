# Контрольная работа (2 семестр, заочная форма)

**Вариант 7** (зачётная книжка 257466 → последние три цифры 466 → 4+6+6 = 16 → 1+6 = 7).

## Задача
Вводится число (пр. 123). Найти число, квадрат которого расположен ближе всего
к введённому числу (пр. 123 → 121 → 11).

## Решение
Файл [`ClosestSquare.java`](ClosestSquare.java). Программа:
1. считывает целое число `n`;
2. берёт приблизительный корень `√n` и проверяет соседние целые числа
   (на случай погрешности вычислений с плавающей точкой);
3. выбирает то целое `k`, у которого `k*k` минимально отличается от `n`;
4. выводит ближайший квадрат, его корень и разницу с введённым числом.

Для `n ≤ 0` ответом является `0`, так как квадраты неотрицательны.

## Запуск
```
javac -encoding UTF-8 ClosestSquare.java
java ClosestSquare
```

## Примеры
| Ввод | Ближайший квадрат | Число | Разница |
|------|-------------------|-------|---------|
| 123  | 121               | 11    | 2       |
| 17   | 16                | 4     | 1       |
| 24   | 25                | 5     | 1       |
| 99   | 100               | 10    | 1       |
| 0    | 0                 | 0     | 0       |
